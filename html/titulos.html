Como utilizar titulos de diferentes tamanhos no html?

Você já parou pra pensar em como utilizar titulos? Este acima por exemplo, que diz"Como utilizar titulos de diferentes tamanhos no html?

Aqui, você vai aprender de uma vez por todas como utilizar esses titulos em tamanhos diferenciados. Mas antes, como se faz um titulo? Para isso, usamos a função h seguida de um número que indica a "prioridade" do titulo, ou seja, seu tamanho e consequentemente o quanto de atenção ele vai puxar do leitor. Veja abaixo, uma compração direta dos titulos h1; h2; h3; h4; h5; e h6.

Este é um titulo h6

Note que quanto maior o numero, menor o titulo. Para usar de fato, você deve incluir em seu código: <h1>Titulo desejado</h1>. Lembrando, quanto maior o número, menor o titulo, sendo o titulo com maior visibilidade o h1.
